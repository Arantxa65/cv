# Curriculum Vitae Project

This project is a professional curriculum vitae webpage designed to present personal information, educational background, and professional experience in a modern and visually appealing format.

## Project Structure

- **index.html**: The main HTML file that contains the structure of the curriculum vitae.
- **css/styles.css**: Contains the styles for the webpage, ensuring a modern and professional design.
- **js/main.js**: JavaScript file for any interactive functionality on the webpage.
- **assets/img**: Directory for storing image files, such as a professional photograph.

## Usage

1. Open `index.html` in a web browser to view the curriculum vitae.
2. Modify `css/styles.css` to change the appearance of the webpage.
3. Use `js/main.js` to add any interactive features or dynamic content as needed.
4. Place any images in the `assets/img` directory for use in the curriculum vitae.
